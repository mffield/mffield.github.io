# Contribution bundle for [`mffield/mffield.github.io`](https://github.com/mffield/mffield.github.io)

This folder contains a proposed content migration from the archived Hugo site into the current Academic Pages / Jekyll site.

Start here:

1. Read [`docs/MIGRATION_SUMMARY.md`](docs/MIGRATION_SUMMARY.md)
2. Read [`docs/GUIDE_TO_MODIFYING_REPOSITORY.md`](docs/GUIDE_TO_MODIFYING_REPOSITORY.md)
3. Copy the replacement files into the repo
4. Remove the demo/template files listed in [`docs/DELETE_OR_REPLACE.md`](docs/DELETE_OR_REPLACE.md)
5. Run the site locally and review the result

The bundle intentionally keeps the structure of the target repository so it can be copied into the repo root.
